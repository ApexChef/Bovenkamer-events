# Profile Report


Kijk, op het moment dat er wat meer profielen gevuld zijn, kunnen we alvast een klein reporting dashboard laten zien van waar iedereen van houdt.

Dus het ligt even aan het type hoe we het presenteren, maar doe eens een voorzet van hoe we de profielen kunnen rapporteren als een dashboard. 

Dan is het belangrijk om te kijken welke secties we kunnen rapporteren.
We kunnen laten zien wat de vaardigheden zijn.
En ook rapporteren op de muziekvoorkeur. 

De JKV-historie:

Hoe lang is iemand gemiddeld lid?

Hoeveel mannen en vrouwen zijn er lid?

Dat komt in de vorige op en voor.

De langste en de kortste.
