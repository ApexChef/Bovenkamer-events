# Feedback

Ik zou graag binnen de app een soort button willen hebben die op elke pagina geladen kan worden om feedback te geven.

Dus die is een klein beetje verborgen, maar misschien met alleen een icoon of een andere manier of een andere weergave. 

Mocht de gebruiker dan op klikken, dan gaat er een iets groter model open, die registreert op welke pagina die zit.

Dus die weet dan ook waar de feedback over gaat. 

En dat je dan misschien een duimpje omhoog of een duimpje naar beneden, of sterretjes kunt geven: één, twee, drie, vier sterren of vijf sterren. 

En dan gelang wat je hebt ingevuld en een kleine beschrijving geeft waarom je het goed vond of waarom je het slecht vond. Of misschien heb je wel een andere suggestie waar je graag zou zien of wat je mist.
